# Roman Numerals Converter Kata
Write a program to convert decimals to Roman numerals. Example:

- roman(1) = "I"
- roman(4) = "IV"
- roman(1954) = "MCMLIV"
- roman(1990) = "MCMXC"

Use the following table to make the correspondence between Roman numerals and decimals:

*Symbol* | *Value*
--- | ---
 I | 1
 V | 5
 X | 10
 L | 50
 C | 100
 D | 500
 M | 1,000
